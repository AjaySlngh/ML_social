# Social Media Metrics

## LinkedIn

### Impressions

The total number of times a post was displayed on users' screens.

A single user may generate multiple impressions if they view the same
post more than once.

------------------------------------------------------------------------

### Engagement Rate

Measures how effectively a post generates interactions relative to its
visibility.

**Formula**

``` text
Engagement Rate = (Total Engagements / Impressions) × 100
```

**Engagements include:**

-   Reactions
-   Comments
-   Reposts
-   Clicks

------------------------------------------------------------------------

### Follower Growth

The increase in the number of followers during a selected time period.

**Formula**

``` text
Follower Growth = Current Followers − Previous Followers
```

or

``` text
Follower Growth (%) = ((Current Followers − Previous Followers) / Previous Followers) × 100
```

------------------------------------------------------------------------

### Clicks

The total number of clicks on a post, excluding engagement actions such
as reactions, comments, and reposts.

Clicks may include:

-   Link clicks
-   Image clicks
-   Document clicks
-   Video clicks
-   "See more" expansion

------------------------------------------------------------------------

## X (Twitter)

### Impressions

The total number of times a post was displayed on X, including
timelines, search results, profile pages, and reposts.

Multiple views by the same user are counted as separate impressions.

------------------------------------------------------------------------

### Engagement Rate

Measures how effectively a post generates interactions relative to its
visibility.

**Formula**

``` text
Engagement Rate = (Total Engagements / Impressions) × 100
```

**Engagements include:**

-   Likes
-   Replies
-   Reposts
-   Link clicks
-   Profile clicks
-   Media clicks
-   Follows generated by the post

------------------------------------------------------------------------

### Reposts

The number of times users shared a post with their followers.

This includes standard reposts (formerly Retweets). Quote posts may be
tracked separately depending on the analytics source.

------------------------------------------------------------------------

### Likes

The total number of users who liked the post.

------------------------------------------------------------------------

### Replies

The number of direct responses posted under the original post.

------------------------------------------------------------------------

# Metric Calculation Rules

## Time Period

All metrics are calculated over the selected reporting period.

------------------------------------------------------------------------

## Engagement Rate

Whenever possible, engagement rate should always be calculated as:

``` text
Engagement Rate = (Total Engagements / Impressions) × 100
```

Using the same formula across LinkedIn and X ensures consistency and
allows direct comparison between platforms.

------------------------------------------------------------------------

## Follower Growth

Follower growth should be computed using the follower count at the
beginning and end of the selected reporting period.

------------------------------------------------------------------------

## Source of Truth

Metrics should always be fetched directly from the official LinkedIn and
X APIs whenever available.
